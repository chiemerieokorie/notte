---
title: "replay"
description: "Get presigned URLs for the session's MP4 replay"
---



**Example:**
```python
replay = session.replay()
replay.download(f"{session.session_id}_replay.mp4")

# Or access the presigned URL directly
print(replay.mp4_url)
```

> Note that the replay is only available after the session has been stopped.

Also you need to perform at least one action for the replay to be available.



## Parameters

<ParamField path="wait" type="bool" default="True">
  If `True`, polls until the replay is ready instead of raising on 404.
</ParamField>

<ParamField path="timeout" type="float" default="240.0">
  Maximum seconds to wait for the replay to become available.
</ParamField>

<ParamField path="poll_interval" type="float" default="5.0">
  Seconds between polling attempts.
</ParamField>

## Returns

[`ReplayResponse`](/sdk-reference/misc/replayresponse): Contains presigned URLs for MP4 download and HLS playlist, plus video metadata.

## Raises

- `ValueError`: If the session hasn't been started yet (no session_id available).
- `ValueError`: If the session is still active — close the session first to generate the replay.
- `TimeoutError`: If the replay does not become available within `timeout` seconds.
